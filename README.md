# SeatApp v1.0 – User Guide

Welcome to SeatApp! This application makes it easy to plan and organize wedding seating arrangements using a simple drag-and-drop interface.

---

## Getting Started

1. Double-click **SeatApp.exe**.
2. If a **guest_list.xlsx** file is included, it will be loaded automatically.
3. If a **seating.json** file is included, your previous seating arrangement will be restored automatically.

---

## How to Use

### Seating Guests

* Select a guest from the list on the left.
* Drag and drop the guest onto any empty seat.
* To move a guest, simply drag them from one seat to another.

### Removing a Guest

* Double-click an occupied seat to remove the guest from that seat.

### Moving Tables

* Click and drag any table to reposition it on the floor plan.
* Table positions are saved automatically.

### Changing the Layout

Use the controls at the top of the window to change:

* Number of round tables
* Seats per round table
* Number of head table seats

Click **Regenerate Layout** to create the new layout.

### Importing a Guest List

Select:

**File → Import Guest List**

Supported formats:

* Excel (.xlsx)
* CSV (.csv)

---

## Saving

SeatApp automatically saves your seating arrangement whenever you make changes.

You can also manually save your current arrangement using:

**File → Save Seating**

To load a previously saved arrangement:

**File → Load Seating**

---

## Keyboard Shortcuts

| Shortcut | Action                 |
| -------- | ---------------------- |
| Ctrl + Z | Undo last seating move |

---

## Files

* **SeatApp.exe** – The application
* **guest_list.xlsx** – Guest list (optional)
* **seating.json** – Saved seating arrangement (optional)

Keep these files together in the same folder.

---

## Tips

* Drag guests directly from the guest list onto seats.
* Drag a seated guest to another seat to move them.
* Regenerating the layout preserves guest assignments whenever possible.
* SeatApp automatically saves your progress after each change.
